# @rah-emil/reraicons

## Usage
*This package supports Vue 3.*

To install the package in your project:
```sh
npm i @rah-emil/reraicons
```

Including an icon inside components:
```vue
<script lang="ts" setup>
import { ArrowUpIcon } from '@rah-emil/reraicons/dist/bold';
</script>

<template>
  <div>
    <ArrowUpIcon />
  </div>
</template>
```
